ASSESSMENT

평가 및 검사

아이의 현재 언어 수준을 정확히 파악해,
꼭 맞는 치료 계획의 출발점이 됩니다.

👶

영아 언어 검사

  • 대상 36개월 미만
  • 소요시간 약 1시간

검사 도구

  • 영유아 언어발달검사 (SELSI)
  • 한국판 맥아더-베이츠 의사소통발달평가 (K-MBCDI)
🧒

유아 언어 검사

  • 대상 36개월 ~ 취학 전
  • 소요시간 약 1시간

검사 도구

  • 수용·표현 어휘력검사 (REVT)
  • 취학 전 아동의 수용·표현언어 발달척도 (PRES)
✅

유아 추가 선택

  • 대상 36개월 ~ 취학 전
  • 소요시간 약 1시간

검사 도구

  • 우리말 조음·음운검사 2 (UTAP2)
📋

검사 안내

  • 표준화된 검사 도구로 아이의 현재 수준을 객관적으로 파악합니다.
  • 검사 결과를 바탕으로 개별화된 치료 계획을 세우고, 보호자께 자세히 안내드립니다.
  • 검사 예약·문의는 언제든 편하게 연락 주세요.
